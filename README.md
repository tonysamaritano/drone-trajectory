# Simple sUA Trajectory Calculator

## How to Use

### Setup

Create .venv folder:

```bash
mkdir .venv
python3 -m venv .venv/
```

Activate Environment and install requirements:

```bash
source .venv/bin/activate
pip3 install -r requirements.txt
```

### Run

After setup, load environment via:

```bash
source .venv/bin/activate
```

Run script:

```bash
python3 trajectory.py
```
